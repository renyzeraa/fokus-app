## Como rodar projeto (windows)

Tenha instalado:

-   Node.js
-   Java JDK
-   Android Studio

Para android studio e java jdk, deve definir corretamente as variaveis de ambiente no windows

1. Clone o projeto e acesse a pasta fokus-app

2. Instale as dependencias

```bash
npm install
```

3. Executar o projeto com expo

```
npm run start
```

4. Ao rodar o projeto no mesmo terminal, ao clicar na tecla "a" ira executar dentro do android studio
